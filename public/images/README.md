# Obrazki do uzupełnienia

W tym folderze powinny znajdować się następujące obrazki:

## Wymagane obrazki:

1. **hero-bg.jpg** - Tło sekcji hero na stronie głównej (np. misy terapeutyczne, spokojny krajobraz)
   - Rozmiar: min. 1920x1080px
   - Format: JPG, zoptymalizowany do web

2. **joanna-session.jpg** - Zdjęcie z sesji terapeutycznej
   - Rozmiar: min. 800x600px

3. **joanna-portrait.jpg** - Portret Joanny na stronę "O mnie"
   - Rozmiar: min. 600x800px

4. **coaching.jpg** - Zdjęcie ilustrujące coaching
   - Rozmiar: min. 600x400px

5. **dzwiekoterapia.jpg** - Zdjęcie mis/gongów
   - Rozmiar: min. 600x400px

6. **szkolenia.jpg** - Zdjęcie z warsztatów
   - Rozmiar: min. 600x400px

7. **sensoryczna-podroz.jpg** - Zdjęcie świec sojowych/warsztatu
   - Rozmiar: min. 600x400px

8. **motanka.jpg** - Zdjęcie lalki motanki
   - Rozmiar: min. 600x400px

## Wskazówki:

- Obrazki powinny być zoptymalizowane do użycia w web (skompresowane)
- Zalecane narzędzia do optymalizacji: TinyPNG, Squoosh
- Zachowaj spójną estetykę - ciepłe, spokojne kolory
- Możesz użyć obrazków z obecnej strony WordPress
